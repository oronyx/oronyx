====

Oronyx (/ˈoʊroʊnɪks) is a POSIX-compatible operating system built from scratch, focusing on performance and modern design principles. 
It aims to balance POSIX compatibility with innovative OS design.

## Building

### Prerequisites

- GNU Make
- x86_64-elf-gcc/g++
- xorriso
- Limine bootloader
- mtools
